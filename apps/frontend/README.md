# 🛒 電商商品管理系統

一個完整的商品管理系統，支援商品的增刪改查功能。

## 功能特色

### 📋 商品列表功能

- ✅ 顯示所有商品列表 (GET /api/products)
- ✅ 響應式卡片設計
- ✅ 商品圖片、名稱、價格、描述、庫存顯示
- ✅ 載入狀態和錯誤處理

### ➕ 新增商品功能

- ✅ 表單新增商品 (POST /api/products)
- ✅ 完整的表單驗證
- ✅ 即時成功/錯誤訊息
- ✅ 自動重新整理商品列表

### ✏️ 編輯商品功能

- ✅ 編輯商品資訊 (PUT /api/products/:id)
- ✅ 彈出式編輯對話框
- ✅ 預填現有商品資料
- ✅ 更新後自動重新整理

### 🗑️ 刪除商品功能

- ✅ 刪除商品 (DELETE /api/products/:id)
- ✅ 確認刪除對話框
- ✅ 安全刪除機制
- ✅ 刪除後自動重新整理

### 🔍 單一商品查詢

- ✅ 查詢單一商品 (GET /api/products/:id)
- ✅ 完整的錯誤處理

## 技術架構

### 前端技術

- **Next.js 15** - React 全端框架
- **TypeScript** - 型別安全
- **Material-UI (MUI)** - 現代化 UI 組件庫
- **響應式設計** - 支援各種螢幕尺寸

### API 路由

- `GET /api/products` - 取得所有商品
- `POST /api/products` - 新增商品
- `GET /api/products/[id]` - 取得單一商品
- `PUT /api/products/[id]` - 更新商品
- `DELETE /api/products/[id]` - 刪除商品

### 組件結構

```
src/app/
├── components/
│   ├── ProductList.tsx      # 商品列表組件
│   ├── AddProductForm.tsx  # 新增商品表單
│   └── EditProductForm.tsx # 編輯商品表單
├── api/products/
│   ├── route.ts            # 商品 CRUD API
│   └── [id]/route.ts       # 單一商品 API
└── page.tsx                # 主頁面
```

## 功能說明

### 1. 商品列表

- 以卡片形式顯示所有商品
- 每個商品卡片包含：圖片、名稱、價格、描述、庫存狀態
- 每個商品都有編輯和刪除按鈕

### 2. 新增商品

- 填寫商品名稱（必填）
- 設定商品價格（必填）
- 輸入商品描述（選填）
- 提供商品圖片網址（選填）
- 設定庫存數量（選填）

### 3. 編輯商品

- 點擊商品卡片的編輯按鈕
- 彈出編輯對話框，預填現有資料
- 修改後點擊更新按鈕

### 4. 刪除商品

- 點擊商品卡片的刪除按鈕
- 確認刪除對話框
- 確認後永久刪除商品
